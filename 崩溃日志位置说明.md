# Bannerlord 崩溃日志位置

## 崩溃日志通常位于以下位置：

### 1. 用户文档目录（最常见）
```
%USERPROFILE%\Documents\Mount and Blade II Bannerlord\CrashReports\
```
或者
```
C:\Users\你的用户名\Documents\Mount and Blade II Bannerlord\CrashReports\
```

### 2. 日志文件目录
```
%USERPROFILE%\Documents\Mount and Blade II Bannerlord\Logs\
```
或者
```
C:\Users\你的用户名\Documents\Mount and Blade II Bannerlord\Logs\
```

### 3. 游戏安装目录
```
D:\SteamLibrary\steamapps\common\Mount & Blade II Bannerlord\Logs\
```

### 4. Mod日志目录（你已经在用的）
```
D:\SteamLibrary\steamapps\common\Mount & Blade II Bannerlord\Modules\ModLogs\
```

## 如何查看崩溃日志

1. **CrashReports文件夹** - 包含详细的崩溃报告（.txt或.log文件）
2. **Logs文件夹** - 包含游戏运行日志
3. **ModLogs文件夹** - 包含mod相关的日志
