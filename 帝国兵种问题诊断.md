# 帝国兵种问题诊断

## 问题描述
用户反馈在游戏中看到：
1. **大量重复的"帝国弓箭手"和"帝国步兵"** - 这些名称不在我们的自定义兵种中
2. **"帝国重装骑兵"、"帝国具装骑兵"、"帝国骑士新兵"** - 这些名称不在我们的自定义兵种中
3. **"自由民新兵"、"高级自由民"、"资深自由民"** - 这些名称不在我们的自定义兵种中
4. **"没穿衣服的帝国重骑兵"** - 装备问题

## 可能的原因

### 1. 原版兵种没有被完全隐藏
- `hide_old_troops.xml` 中可能遗漏了某些原版兵种
- 原版兵种使用了相同的string id（如`{=LbjxH9yp}`），导致多个兵种显示相同名称

### 2. 其他Mod的兵种在干扰
- 可能有其他mod也在加载帝国兵种
- 这些mod的兵种名称与我们的不同

### 3. 加载顺序问题
- `hide_old_troops.xml` 在 `imperial_troops_new.xml` 之前加载（正确）
- 但可能其他mod在之后加载，覆盖了我们的隐藏设置

### 4. String ID冲突
- 原版兵种使用相同的string id，导致显示名称重复
- 需要确保所有原版兵种都被隐藏

- T2: `imperial_t2_eques_auxilia` - "帝国辅助骑兵"
- T3: `imperial_t3_lancearius_eques` - "帝国长枪骑兵"
- T3: `imperial_t3_eques_sagittarius` - "帝国弓骑兵"
- T4: `imperial_t4_lancearius_veteranus` - "帝国老练长枪骑兵"
- T4: `imperial_t4_eques_sagittarius_veteranus` - "帝国老练弓骑兵"
- T5: `imperial_t5_lancearius_cataphractarius` - "帝国重装长枪骑兵"
- T5: `imperial_t5_eques_sagittarius_praetorianus` - "帝国历战弓骑兵"

### 贵族骑兵线：
- T3: `imperial_t3_cataphract` - "帝国具装骑兵"
- T4: `imperial_t4_heavy_cataphract` - "帝国重装具装骑兵"
- T5: `imperial_t5_elite_cataphract` - "帝国精锐具装骑兵"

**注意：我们的兵种中没有"帝国重装骑兵"、"帝国骑士新兵"、"自由民"等名称！**

## 需要检查的项

1. **检查hide_old_troops.xml是否隐藏了所有原版帝国兵种**
2. **检查是否有其他mod在加载帝国兵种**
3. **检查原版兵种的string id，确保它们都被隐藏**
4. **检查"没穿衣服的帝国重骑兵"是哪个兵种，修复其装备**
