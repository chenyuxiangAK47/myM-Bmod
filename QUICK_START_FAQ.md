# 快速开局功能常见问题

## Q: 在主菜单看不到"快速开始"按钮？

**A:** 当前版本的快速开局功能**没有主菜单按钮**，它是**自动触发**的。

### 如何使用：

1. 在主菜单选择 **"新战役"** 或 **"沙盒模式"**
2. 完成角色创建流程
3. 进入游戏后，**自动获得 100,000 金币**

### 为什么没有按钮？

当前实现使用的是标准 API，无法：
- 在主菜单添加按钮
- 跳过角色创建流程

### 如果需要主菜单按钮

需要使用 **Harmony 补丁**来：
1. 修改主菜单界面，添加"快速开始"按钮
2. 跳过角色创建流程
3. 使用默认角色数据直接开始游戏

这需要：
- 安装 `Bannerlord.Harmony` 模组
- 编写 Harmony 补丁代码
- 修改主菜单 UI

---

## Q: 如何确认快速开局功能已启用？

**A:** 检查 `SubModule.xml` 中的 `SubModuleClass` 是否已取消注释：

```xml
<SubModuleClass>NordThrowingAxeMod.WelcomeMessageSubModule</SubModuleClass>
```

如果这行被注释了（`<!-- ... -->`），快速开局功能就不会工作。

---

## Q: 可以自定义快速开局的金币数量吗？

**A:** 可以！修改 `SubModule\WelcomeMessage.cs` 中的这一行：

```csharp
Campaign.Current.AddMoneyToPlayerParty(100000);  // 修改这里的数字
```
