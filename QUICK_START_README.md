# 快速开局功能说明

## 功能概述

本mod在每次新战役开始时自动给玩家 **100,000 金币**，方便测试和快速开始游戏。

## 实现方式

- 使用 `OnCampaignStart` 钩子在战役开始时自动执行
- 无需手动操作，进入游戏后自动获得金币
- 每次新开战役都会获得金币

## 注意事项

1. **跳过角色创建**：由于Bannerlord的API限制，完全跳过角色创建流程（选择文化、难度、出生等）需要更复杂的实现，可能需要：
   - 使用 Harmony 来hook菜单系统
   - 使用 UIExtenderEx 来添加自定义菜单按钮
   - 或者使用现有的 QuickStart mod（Nexus Mods ID: 2524）

2. **当前实现**：目前只实现了自动给金币的功能，角色创建流程仍需手动完成。

3. **编译要求**：需要编译C#代码生成DLL，并在 `SubModule.xml` 中取消注释 `SubModuleClass` 行。

## 未来改进

如果需要完全跳过角色创建，可以考虑：
- 集成现有的 QuickStart mod
- 使用 Harmony 来修改游戏启动流程
- 使用 UIExtenderEx 添加"快速开局"菜单按钮
