# 快速开局功能使用指南

## 功能说明

本模组已实现快速开局功能，会在每次新战役开始时自动给予玩家 **100,000 金币**，方便测试和快速开始游戏。

## 使用方法

1. **启动游戏**，进入主菜单
2. 选择 **"新战役"** 或 **"沙盒模式"**
3. 正常完成角色创建流程（选择文化、难度、出生背景等）
4. 进入游戏后，会自动收到 **100,000 金币** 和提示消息

## 注意事项

- 每次开始新游戏时都会自动给予金币
- 金币会在角色创建完成后立即发放
- 如果不想使用此功能，可以注释掉 `SubModule.xml` 中的 `SubModuleClass` 行

## 技术实现

快速开局功能通过 `SubModule\WelcomeMessage.cs` 实现，在 `OnCampaignStart` 方法中自动给玩家金币。

## 未来改进

如果需要完全跳过角色创建流程，需要更复杂的实现，可能需要：
- 使用 Harmony 补丁修改游戏启动流程
- 创建自定义的游戏模式
- 修改主菜单界面（需要 UI 扩展模组支持）

目前实现的是"半自动"快速开局：仍然需要完成角色创建，但会自动给予测试资金。
